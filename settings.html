About TradingView PiP

This extension allows you to keep TradingView charts visible in a floating overlay window while working on other tasks.

Features

  • Real-time chart capture from TradingView tabs
  • Customizable area selection for focused viewing
  • Floating overlay window that stays on top
  • Adjustable window size and position
  • Low CPU usage with optimized capture

How to Use

  1. Navigate to any webpage in any tab
  2. Click the extension icon and select "Select Content Area"
  3. Drag to select the content region you want to monitor
  4. Click "Start Picture-in-Picture" to begin capturing
  5. The floating window will show your selected content area
